The Art of Asking

How to talk to a robot

Bad prompt

Write me an email about the QBR.

Good prompt

You are an account manager at a travel management company. Write a professional email to my client Art at Vanderlay Industries inviting him to our Q4 QBR on 15 Feb. Tone: warm but professional. Include agenda: spend overview, policy compliance, savings, airline program review. Under 200 words.

The golden rules

  • Be specific — who, what, why, how long
  • Give context — "I'm an AM at a TMC"
  • Set the tone — professional, casual, formal
  • Set constraints — word count, format, audience
  • Iterate — "make it shorter" / "add more detail"

Think of it like briefing a new team member.
Better brief = better output.

How it works

How these get built

The process:

  • Describe the problem in plain English
  • Claude writes the HTML/JavaScript
  • Test it, tell it what's wrong
  • It fixes it. Repeat until done.
  • End result: a standalone HTML file
